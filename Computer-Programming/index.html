Computer Programming

With an abundance of screen time during COVID, I picked started programming. Most of my projects have programed on Scratch, a software designed by MIT that allows the user to create programs without typing out code. Blocks that represent code functions snap together in a visual workspace. The code blocks have the same functions you'd encounter in programming languages such as Python or Javascript.

Mathmatics

Raycasting Navigation

An entity shoots out rays similar to a LIDAR system and then analyzes the data to path-find its way around any environment.

Collatz Conjecture Visualization

After watching a video about the Collatz Conjecture, I made this program to simulate and visualize it.

Abstract Image Generater

Choose from three generation models to generate random but patterned images at various resolutions.

Line Point Collision Check

This project is revolutionary for me as it can check if a coordinate intersects any line segment of any thickness.

Triangle Collision Check

Using some pretty complicated math, this program can detect if one coordinate is within the triangle made by three other coordinate points.
